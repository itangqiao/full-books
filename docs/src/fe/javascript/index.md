## 数据类型

### 原始数据类型 （7个）
`number`、`boolean`、`string`、`null`、`undefined`、`symbol`、`BigInt` 

### 引用类型 (对象类型)
`Function`、`Object`、`Array`、`Date`、`RegExp`  等

### 假值 falsy（6个）
`false`、`null`、`0`、`""`、`undefined` 、`NaN`  

::: tip
console.log( null == undefined ) // true
:::
